# 🕹️ OpenShift Tetris
A lightweight Tetris clone built with Vanilla JavaScript and HTML5 Canvas.

## How to Play
1. Clone the repo.
2. Open `index.html` in any browser.
3. Use **Arrow Keys** to move and **Q/W** to rotate.
4. Use **Spacebar** to quick drop.
5. Use **P** to Pause the game.

## Project Structure
- `index.html`: The UI structure.
- `style.css`: The dark-mode styling.
- `tetris.js`: The game engine logic.
