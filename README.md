# 🕹️ OpenShift Tetris
A lightweight Tetris clone with Score Database.

## Added Features
1. Added Top 5 Score Board
2. Added Particle Effects on Hard Drop
3. Warning Notification
4. Combo widgets
5. Score calculation update

## Project Structure
- `index.html`: The UI structure.
- `style.css`: The dark-mode styling.
- `tetris.js`: The game engine logic.
